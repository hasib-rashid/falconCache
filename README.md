### Hosting the Bot

-   Rename .env.example to .env
-   Fill the proper info in .env
-   Run `npm install` or `yarn` to install all the required Dependencies
-   Run `tsc` and have `typescript` to compile the typescript
-   Run `node .` to run the bot
